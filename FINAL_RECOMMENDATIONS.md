**Что нужно учесть:**

### РЕШЕНИЕ 2: Когда нанять Sales Manager?

**Варианты:**
- Q1 (январь-март) — рано, но снимет нагрузку с вас
- Q2 (апрель-июнь) — оптимально ⭐ **РЕКОМЕНДУЕТСЯ**

---

### РЕШЕНИЕ 3: Запускать ли Flipping?

**Условия запуска:**
- ✅ Есть конкретный объект с потенциалом +30% к стоимости
- ✅ Есть €100k–200k свободных средств
- ✅ Есть проверенные подрядчики (ремонт, дизайн)

**Если хотя бы одно условие не выполнено:**
- ❌ Отложить на Q3 или 2027

**Рекомендация:** Начать искать объекты в Q1, но запускать только если все условия выполнены к Q2.

---

### РЕШЕНИЕ 4: Нужна ли автоматизация в Q3?

**Условия запуска:**
- Команда >10 агентов (онбординг и образование имеют смысл)
- Документооборот занимает >10 часов/неделю (автоматизация окупается)

**Если условия не выполнены:**
- Отложить на 2027 (преждевременная оптимизация)

**Рекомендация:** Решить в конце Q2, когда будет понятен масштаб команды.

---

### РЕШЕНИЕ 5: Модульные дома — да или нет?

**Честный ответ:** НЕТ (в 2026).

**Причины:**
- Требует отдельной команды (2–3 человека full-time)
- Требует больших инвестиций (€200k–500k)
- Отвлечёт от текущего бизнеса

**Если очень хочется:**
- Найти партнёра, который будет вести проект полностью автономно

**Рекомендация:** Отложить на 2027 или делегировать полностью.

### Расходы на команду и маркетинг (Сценарий 2):

| Статья | Месяц | Год |
|--------|-------|-----|
| **Зарплаты:** | | |
| Маркетолог | €3,500 | €42,000 |
| Sales Manager (с Q2) | €2,500 (9 мес) | €22,500 |
| **Маркетинг:** | | |
| Реклама (Google/Meta) | €3,000 | €36,000 |
| Разработка лендингов | €3,000 (единоразово) | €3,000 |
| **Подрядчики:** | | |
| Разработка и автоматизация | €2,000 | €24,000 |
| Видеограф/монтажёр | €1,000 | €12,000 |
| Cold calling агентство | €2,000 (с Q2) | €18,000 |
| HR-рекрутер | €1,500 | €18,000 |
| **ИТОГО** | **€12,500–15,000** | **€175,500** |

### Целевая выручка для окупаемости:

**Минимум (Break-even):**  
€175,500 расходов + €100,000 операционка = €275,500 нужно заработать

При марже 3–5% в real estate:
- **€275k выручка = €5.5M–9.2M в сделках за год**
- **€23k/месяц выручка = €460k–770k в сделках/месяц**

**Комфортно (Прибыль €100k+):**
- **€400k выручка = €8M–13M в сделках за год**
- **€33k/месяц выручка = €660k–1.1M в сделках/месяц**

**Вопрос:** Реально ли закрывать €500k–1M в сделках каждый месяц?
- Если да → план выполним
- Если нет → снизить расходы (меньше рекламы, отложить часть проектов)
