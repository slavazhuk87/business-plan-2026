# ROI / VALUE CASE (Q1, 90 ДНЕЙ) — ЧТО ПОЛУЧАЮТ БОССЫ

## 1) Executive take (в 30 секунд)

**Мы просим бюджет на 90 дней, чтобы получить 3 вещи:**
1) **Рост выручки** за счёт: buyers (inbound), listings, девелоперы (new developments), партнёрства (USA/UK/PL/EU + тест Cyprus).
2) **Систему контроля**: ни один лид/листинг/контакт не теряется, есть SLA и воронка.
3) **Employer brand**: агентство становится медийным → проще нанимать сильных агентов → они приносят листинги.

**Комиссия:** вы указали ориентир **1.25%** с каждой сделки (минимум).

---

## 2) Данные рынка (для ориентира)

Мы не опираемся на “ощущения”, а на публичные ориентиры:
- В 2024 году в Португалии было продано порядка **156,325 объектов недвижимости** (оценка, встречается в рыночных обзорах и публикациях).  
  Источник (пример): The Portugal News упоминает *“Portugal selling 156,325 total properties in 2024”*.

> Для презентации важно: даже **очень маленькая доля рынка** даёт заметные деньги, если есть система лидов+контента.

---

## 3) Формула (простая и CFO-friendly)

**Выручка = Кол-во сделок × Средняя цена сделки × 1.25%**

Чтобы не спорить о “средней цене”, считаем таблицей для 3 уровней цены сделки:
- €250k (консервативно)
- €350k (реалистично для многих сделок в Лиссабоне/Порту)
- €500k (апсайд сегмент)

---

## 4) Сценарии по доле рынка (годовые ориентиры → Q1)

Берём **N = 156,325 продаж/год** и рассматриваем доли рынка:
- **0.01% рынка** → 16 сделок/год
- **0.02% рынка** → 31 сделка/год
- **0.05% рынка** → 78 сделок/год

Для Q1 (90 дней) делим примерно на 4:
- 0.01% → ~4 сделки / Q1
- 0.02% → ~8 сделок / Q1
- 0.05% → ~20 сделок / Q1

**Важно:** это не “обещание”, а ориентир масштаба, который становится достижимым при медийности + LeadOps + партнёрства + листинги + девелоперы.

## 6) Как эти сделки “добываются” (4 источника выручки)

### 6.1 Buyers (inbound: IG/YT/TG + вебинары)
- Воронка: охват → заявки → квалификация → показы → оффер
- Драйверы Q1: **контент-конвейер + 3 вебинара**

### 6.2 Listings (seller-side)
- Воронка: база → прозвон/контакт → встреча → эксклюзив/листинг → продажа
- Драйверы Q1: **прозвоны (AI+человек) + медийность (доверие)**

### 6.3 New Developments (девелоперы)
- Воронка: девелопер → пилот → “sales point” → лиды/показы/резервы
- Драйвер Q1: **ваши переговоры + 1 пилот-проект**

### 6.4 Партнёрства (USA/UK/PL/EU + тест Cyprus)
- Воронка: партнёр → интро → консультация → сделка
- Драйвер Q1: **10–15 встреч → 5 активных партнёров**

---

## 7) “Если сделки длинные” — как доказать прогресс за 90 дней

Боссам важно видеть управляемую систему, даже если закрытия часть уйдёт в Q2.

### KPI-результаты Q1, которые должны быть (без “ванити”)
- 100% лидов в системе, SLA (первый контакт) ≤ 10 минут
- отчёт Leads → Qualified → Meetings → Deals каждую неделю
- 3 вебинара: registrations / attendance / booked calls / pipeline created
- 5 активных партнёров + первые интро
- 1 девелоперский пилот: лиды/показы/резервы
- по листингам: конверсия база → контакт → встреча → листинг (фиксируем цифры)

---

## 8) Почему это “выгодно боссам”

**Они покупают не маркетинг, а системный рычаг:**
- бренд → лиды → сделки
- бренд → агенты → листинги → сделки
- партнёрства → качественные клиенты
- девелоперы → доступ к продукту + рост статуса
- LeadOps → исчезают потери и хаос

---

## 9) Критическая точка (день 90) — Go/No-Go

### GO (масштабируем в Q2)
Если:
- система лидов работает (SLA/учёт)
- контент выходит стабильно (конвейер живой)
- есть рост воронки (встречи/показы/оферы)
- девелоперский пилот начался (или подписан LOI)

### NO-GO (пересматриваем)
Если:
- лиды теряются, SLA не соблюдается
- нет стабильного выпуска контента
- нет прогресса в воронке после 60–90 дней
